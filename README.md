# IND-MONEY-RAG-CHATBOT
THIS IS A RAG CHATBOT WHICH TAKES DATA FROM IND MONEY WBESITE FOR SBI MUTUAL FUND (AMC).

Mutual Fund FAQ Assistant (Facts-Only)

Product Context: INDMoney
AMC Used: SBI Mutual Fund

This project builds a facts-only Mutual Fund FAQ assistant that answers common mutual fund scheme questions using official public sources only.

The assistant provides factual answers about:

Expense ratio

Exit load

Minimum SIP

ELSS lock-in period

Riskometer level

Benchmark index

How to download mutual fund statements

Each answer:

Is ≤ 3 sentences

Includes one official source link

Shows “Last updated from sources”

Provides no investment advice

The goal is to simulate a knowledge assistant similar to what could exist inside the INDMoney platform to help retail investors quickly retrieve verified mutual fund information.
