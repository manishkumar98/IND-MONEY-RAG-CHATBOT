# Disclaimer

The following snippet is displayed in the UI of the SBI MF RAG Assistant:

> [!WARNING]
> This chatbot is an experimental tool for information purposes only. It is **not** a financial advisor. All responses are derived from public documentation. Mutual fund investments are subject to market risks; read all scheme-related documents carefully.

## Core Rules
1. **Facts-Only**: The assistant only provides factual information about the AMC and its schemes. It refuses to provide investment advice, predictions, or recommendations.
2. **PII Protection**: Any PII (Personal Identifiable Information) such as PAN, Aadhaar, Bank Details, etc., will be flagged and the assistant will refuse to process it.
3. **No Financial Advisory**: The assistant will explicitly state that it cannot provide financial advice and recommend consulting a human advisor for personalized recommendations.
4. **Attribution**: All factual answers include a "Last updated from sources" section with clickable links and the date the information was retrieved.
